# Sumit Phuyal — Portfolio

A mesmerizing, animated personal portfolio website built with plain **HTML, CSS, and JavaScript** (no frameworks).

🔗 **Live site:** _add your URL here after deploying_

## Features

- Animated SVG preloader, particle constellation background, and aurora blobs
- Custom cursor, scroll-progress bar, and scroll-reveal animations
- Auto-typing role text, count-up stats, and animated skill bars
- 3D tilt hero card with a circular profile photo
- Dark / light theme toggle (remembers your choice)
- Fully responsive, with `prefers-reduced-motion` accessibility support
- Working contact form (opens your email client, no backend required)

## Sections

Home · About · Skills · Experience · Projects · Education & Certifications · Contact

## Run locally

Just open `index.html` in a browser, or serve the folder:

```bash
python -m http.server 8000
# then visit http://127.0.0.1:8000
```

## Structure

```
.
├── index.html          # markup
├── css/style.css       # styling + animations
├── js/script.js        # interactivity
└── assets/             # profile photo + resume PDF
```

## Deploy

This is a static site, so it works on any static host:

- **GitHub Pages:** push this repo to GitHub, then Settings → Pages → deploy from `main` / root.
- **Netlify:** drag this folder onto <https://app.netlify.com/drop>, or connect the repo.
- **Vercel:** import the repo at <https://vercel.com/new>.
